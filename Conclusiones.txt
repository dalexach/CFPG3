Con el desarrollo de este proyecto hemos conseguido crear un programa que genera archivo de texto plano con información de vendedores y productos generados de manera parcialmente aleatoria, cumpliendo con los requisitos principales especificados en el instructivo de entregas. Adicionalmente, gracias a este proyecto pusimos a prueba nuestras habilidades de programación, documentación y uso adecuado de repositorios grupales.
El código fue desarrollado siguiendo prácticas correctas y eficientes a la hora de realizar cambios y revisiones grupales, lo cual permitió cumplir exitosamente con las fechas de entrega.
